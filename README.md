# my little corner 🌿

a small personal website — part dashboard, part diary, part digital room.

## ✨ about

this project is an attempt to create a space on the internet that feels more *human* than a traditional portfolio.

instead of focusing only on projects or technical skills, this site reflects:

* what i'm building
* what i'm listening to
* what i'm playing
* what i'm thinking about
* and the small things i enjoy

it's meant to feel like a quiet, cozy corner of the internet.

## 🧠 concept

inspired by indie web and personal homepages, this project moves away from:

* corporate-style portfolios
* productivity dashboards

and towards something more:

* personal
* introspective
* creative
* slightly playful

## 🛠 tech

* React (Vite)
* deployed with GitHub Pages (via GitHub Actions)
